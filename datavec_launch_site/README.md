# DataVec launch website V2

Static website prototype based on the updated DataVec brand direction.

Public product names:

- PRYZM: JavaScript optimization engine
- PÜLS: Persistent runtime engine

Pricing model included:

- Free: $0 per month
- Basic: $29 per month plus usage
- Pro: $199 per month plus usage
- Enterprise: custom

Billing strategy:

- Monthly plan fee
- Included usage
- Transparent usage after included limits
- Alerts, budgets, and hard caps

Notes:

- No internal technology names are used on the website.
- PRYZM and PÜLS are capitalized in visible website copy.
- Dashboard and metric examples are labelled as demo data.
- The contact form is a visual mockup and needs backend integration before launch.
